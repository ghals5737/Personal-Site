## Coroutine이란?

Coroutine은 비동기 프로그래밍을 동기 코드처럼 작성할 수 있게 해주는 Kotlin의 강력한 기능입니다.

### 기존 방식의 문제점

```kotlin
// Callback Hell
fun fetchUserData(userId: Long, callback: (User) -> Unit) {
    userRepository.findById(userId) { user ->
        orderRepository.findByUserId(user.id) { orders ->
            paymentRepository.findByOrders(orders) { payments ->
                callback(user.copy(orders = orders, payments = payments))
            }
        }
    }
}
